LANGKAH 6
Sambungkan GitHub → HF Space
HF otomatis baca kode dari GitHub dan build server · Bisa tanpa terminal!
1
Cara A — Lewat web HF (tanpa terminal, paling mudah)
🌐
Di halaman Space kamu → klik tab "Files" → klik "Add file" → "Upload files"

LANGKAH 7
OpenAI Key + Set Secret di HF
Dua sub-langkah: dapat key → masukkan ke HF aman
Bagian A — Dapat OpenAI API Key
A
Daftar & buat key di OpenAI
🌐
Buka platform.openai.com → Sign up (atau Login)
🎁
Akun baru otomatis dapat $5 kredit gratis = 833 menit Whisper tanpa bayar
🔑
Klik nama profil → API Keys → Create new secret key → Name: NurAI → Create
📋
Copy key: sk-proj-xxxxxxxxxx → simpan di Notepad. Hanya ditampilkan SEKALI!
Bagian B — Masukkan Key ke HF Space (dengan aman)
B
Set Secret di HF Space
⚙️
Buka Space kamu → tab Settings → scroll ke "Variables and secrets"
➕
Klik New secret → isi:
Name
OPENAI_API_KEY ← huruf besar semua, persis ini
Value
paste sk-proj-xxxxx kamu di sini
💾
Klik Save → Space restart otomatis ~30 detik
🔐Secret tersimpan terenkripsi di server HF. Tidak ada yang bisa lihat nilainya, termasuk kamu sendiri setelah disimpan. Aman!

LANGKAH 10 · FIREBASE
Firebase — Aktifkan Google Login
5 klik dan Google Login siap
1
Aktifkan Authentication
🔐
Di Firebase Console → menu kiri klik Authentication → klik Get started
📋
Klik tab Sign-in method → klik Google di daftar provider
🔘
Toggle Enable → isi Project support email (email kamu) → klik Save
✅Google Login sekarang aktif! Pengguna bisa login dengan akun Google mereka.
2
Tambahkan domain Netlify ke Authorized Domains
Ini WAJIB agar Google Login bisa dipakai dari URL Netlify kamu.
⚙️
Di Authentication → tab Settings → scroll ke Authorized domains
➕
Klik Add domain → masukkan domain Netlify kamu: xxxx.netlify.app → klik Add

Lakukan ini lagi untuk domain HF Space: USERNAME-nurai-whisper.hf.space
⚠️Kalau domain belum ditambahkan, Google Login akan error "auth/unauthorized-domain". Ini penyebab #1 login gagal!

LANGKAH 11 · FIREBASE
Firebase — Buat Database Firestore
Tempat skor tajwid & data pengguna tersimpan di cloud
1
Buat Firestore Database
🗄️
Firebase Console → menu kiri klik Firestore Database → klik Create database
🔒
Pilih Start in test mode → klik Next
Test mode artinya semua orang bisa baca/tulis — OK untuk development. Nanti bisa diubah ke production rules.
🌏
Pilih region: asia-southeast2 (Jakarta) untuk koneksi tercepat dari Indonesia → klik Enable
⏳
Tunggu ~30 detik → Firestore siap! Akan terlihat halaman database kosong.
2
Struktur data yang akan tersimpan otomatis
📁
users/{uid}
nama, email, lastLogin
📁
scores/{uid_surah_ayah}
skor per ayat per pengguna
💡Kode NurAI sudah otomatis menyimpan ke struktur ini setiap kali pengguna login atau rekam ayat.

LANGKAH 14
Deploy NurAI ke Netlify
Drag & drop — 30 detik dapat URL publik dengan HTTPS
⚠️Mikrofon hanya bisa diakses dari https:// — tidak bisa dari file lokal. Wajib deploy ke Netlify dulu sebelum test rekam suara.

Upload ke Netlify Drop (tanpa daftar)

• whisper-client-hf.js

Drag & drop kedua file ke area drop di netlify.com/drop — tunggu 10–30 detik
🎉

Tambahkan URL Netlify ke Firebase Auth
🔥
Firebase Console → Authentication → Settings → Authorized domains → Add domain → masukkan random-name.netlify.app
✅Setelah ini Google Login akan berfungsi dari URL Netlify kamu.
